# echos-eden
Le Contexte : Une Urgence Biodiversité
Le projet s'inscrit dans un constat alarmant établi par l'ONU via le rapport IPBES de 2019:  

Environ 1 million d'espèces sur les 8 millions estimées sont aujourd'hui menacées d'extinction.  

Une grande partie de cette biodiversité demeure mystérieuse, puisque seulement 10 à 20 % des espèces ont été décrites par la science à ce jour.  

Le Concept : "Echos d’Eden"
Pour répondre à cet enjeu, "Echos d’Eden" propose une immersion dans un monde virtuel inhabité. L'expérience se décline comme suit :  


Exploration temporelle et climatique : Durant une balade de 2 à 3 minutes, l'utilisateur peut changer de saison, accélérer le temps ou modifier la météo.  


Rencontres animalières : Le parcours permet de croiser des animaux familiers mais aussi des espèces inconnues, offrant une vision complète de l'environnement.  

L'Objectif : Sensibiliser par l'Immersion
Bien plus qu'une simple prouesse technologique, ce projet se veut être une véritable sonnette d'alarme. L'objectif est de susciter une prise de conscience individuelle sur la fragilité de notre écosystème et sur ce que représente réellement la perte de la biodiversité.
